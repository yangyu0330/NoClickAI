# NoClick AI

NoClick AI는 사용자가 앱 메뉴를 직접 누르지 않고 자연어 목적을 입력하면 실행 계획, 위험도별 승인, 자동화 로그를 한 흐름으로 관리하는 MVP입니다.

## 구현 범위

- Android 브라우저와 데스크톱 브라우저에서 사용할 수 있는 반응형 PWA
- 자연어 목적 입력과 대표 예시 3개
- AI 실행 계획 카드: 목표, 마감일, 필요한 앱, 단계, 위험도
- 개인 API Key 입력 UI: 코드에 키를 하드코딩하지 않고 브라우저 로컬 저장소에만 저장
- OAuth 연결 데모: Google Calendar, Gmail, Notion, Slack, Discord, Browser Agent 연결 상태 토글
- 위험도별 승인 정책
  - 낮음: 자동 실행 가능
  - 승인: 실행 전 승인 필요
  - 개별 승인: 메시지 발송, 제출 폼 입력 같은 책임 행동은 개별 승인
  - 차단: 송금, 결제, 계정 변경은 MVP에서 실행하지 않음
- 실행 로그와 완료 히스토리

## 데모 시나리오

1. `다음주 수요일 4시까지 계획서 제출`
2. `다음주 목요일 9시까지 통합회의`
3. `이번주 수요일까지 팀원들에게 내가 부여한 과제를 모두 해오도록 다시 공지`

## 로컬 실행

```bash
npm install
npm run dev
```

빌드 확인:

```bash
npm run build
```

## 실제 연동으로 확장할 지점

현재 외부 앱 실행은 안전한 데모 모드로 시뮬레이션합니다. 실제 서비스화 단계에서는 아래 어댑터를 서버 API 뒤에 붙이면 됩니다.

- Google Calendar API: 일정 조회, 일정 생성
- Gmail API: 메일 초안 생성, 사용자 승인 후 발송
- Notion API 또는 Google Docs API: 할 일 보드와 문서 초안 생성
- Slack/Discord API: 메시지 초안, 예약 발송
- Playwright 기반 Browser Agent: 제출 폼 입력, 최종 제출 전 사용자 승인

민감 작업인 결제, 송금, 개인정보 대량 전송, 삭제 작업은 기본 차단 정책을 유지합니다.
